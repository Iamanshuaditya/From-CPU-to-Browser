# Challenge: Add Profiler Instrumentation

**Difficulty:** medium

## Objective
Instrument one project pipeline with actionable stage-level metrics.

## Scope
- scoped timers per stage
- counters for key events
- structured report output

## Acceptance Criteria
- [ ] At least 5 stages instrumented with timing.
- [ ] Metrics output includes p50/p95 values.
- [ ] Includes one bottleneck analysis and fix proposal.
- [ ] Instrumentation overhead is documented.
- [ ] Adds regression guard to keep metrics from breaking silently.

## Submission Requirements
- instrumentation design note
- example output
- before/after profiling comparison
