# Challenge: Build a DevTools-like Timeline Viewer

**Difficulty:** hard

## Objective
Build a simple timeline UI that visualizes recorded stage events over time.

## Scope
- ingest event trace JSON
- render lane-based timeline
- zoom/filter by stage
- highlight long tasks and frame budget violations

## Acceptance Criteria
- [ ] Timeline displays events with start/end duration.
- [ ] Supports filtering by stage and severity.
- [ ] Highlights events exceeding threshold (e.g., >16ms frame budget).
- [ ] Includes sample traces from at least two projects.
- [ ] Includes documentation for trace schema and usage.

## Submission Requirements
- trace schema doc
- timeline screenshot/demo
- note on known limitations and next improvements
