# Challenge: Implement CSS Grid Subset

**Difficulty:** hard

## Objective
Add a practical CSS Grid subset to the tiny layout engine.

## Scope
- explicit track definitions
- item placement by row/column
- spanning support
- basic track sizing

## Acceptance Criteria
- [ ] Supports fixed and fractional track units (subset documented).
- [ ] Handles item spanning with deterministic placement.
- [ ] Produces stable layout snapshots for test corpus.
- [ ] Includes at least 15 grid-specific tests.
- [ ] Includes complexity/performance notes for placement algorithm.

## Submission Requirements
- implementation notes
- test evidence
- one benchmark comparing before/after layout cost
