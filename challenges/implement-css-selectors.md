# Challenge: Implement CSS Selectors Engine

**Difficulty:** medium

## Objective
Add robust selector matching and specificity handling to the style engine.

## Scope
- type, class, id selectors
- descendant and child combinators
- specificity calculation and tie-breaking

## Acceptance Criteria
- [ ] Selector parser handles valid/invalid inputs safely.
- [ ] Matching correctness for nested DOM structures.
- [ ] Specificity and source-order tie-breaks behave correctly.
- [ ] Includes at least 25 selector/cascade tests.
- [ ] Performance note for matching complexity is included.

## Submission Requirements
- selector grammar summary
- test evidence
- profiling note on matching hot path
