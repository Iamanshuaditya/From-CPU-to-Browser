# HTML+CSS Renderer Rubric

Scoring: 100 points total.

## Parser Correctness (25)
- HTML parser robustness (15)
- CSS parser robustness (10)

## Style System Correctness (25)
- Selector matching accuracy (10)
- Specificity and cascade behavior (15)

## Layout Correctness (20)
- Box model calculations (10)
- Block/inline layout behavior (10)

## Testing and Debuggability (20)
- Snapshot and regression tests (10)
- Useful debug dumps and tracing (10)

## Engineering Quality (10)
- Code structure and docs quality

Pass thresholds:
- Minimum completion: >= 70
- Mastery completion: >= 85
