# Project: Mini Browser Pipeline (Parse -> Style -> Layout -> Paint)

Build a vertical slice of a browser rendering engine and make every stage measurable.

## Scope
- HTML parse
- CSS parse + cascade
- layout (block + optional flex/grid subset)
- paint list generation
- simple compositing model

## Deliverables
- executable rendering pipeline
- stage timing instrumentation
- correctness tests per stage
- performance checkpoint report
