# Mini Browser Milestones

## Milestone 1: Parse Layer
- HTML + CSS parser baseline.

Exit criteria:
- parser test suite passes.

## Milestone 2: Style Layer
- selector matching and computed styles.

Exit criteria:
- cascade tests pass.

## Milestone 3: Layout Layer
- layout tree and geometry output.

Exit criteria:
- layout snapshots stable.

## Milestone 4: Paint Layer
- display list generation and frame output.

Exit criteria:
- visual or command snapshot correctness.

## Milestone 5: Metrics and Optimization
- stage timing instrumentation + one optimization pass.

Exit criteria:
- before/after report with measurable gain.
