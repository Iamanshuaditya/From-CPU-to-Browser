# Mini Browser Rubric

Scoring: 100 points.

## Pipeline Correctness (40)
- parse correctness (10)
- style/cascade correctness (10)
- layout correctness (10)
- paint/composite correctness (10)

## Instrumentation and Analysis (20)
- stage timing quality (10)
- analysis quality (10)

## Robustness (15)
- malformed input handling
- deterministic output behavior

## Testing (15)
- snapshot and regression tests

## Engineering Quality (10)
- modular architecture + documentation

Pass thresholds:
- Minimum completion: >= 70
- Mastery completion: >= 85
