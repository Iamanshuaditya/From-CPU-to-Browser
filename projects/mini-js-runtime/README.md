# Project: Mini JavaScript Runtime (Event Loop + GC Basics)

Build a minimal JavaScript runtime that can parse, execute, schedule tasks, and collect memory.

## Scope
- parser + AST (subset)
- bytecode VM or interpreter
- closures/prototype lookup basics
- event loop (microtask/macrotask ordering)
- mark-and-sweep GC baseline

## Output
- runtime executable with script runner
- event loop ordering tests
- GC metrics and leak checks
- profiling notes
