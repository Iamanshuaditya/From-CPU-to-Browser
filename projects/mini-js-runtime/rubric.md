# Mini JS Runtime Rubric

Scoring: 100 points.

## Language + Execution Correctness (35)
- parser correctness (10)
- runtime semantics (15)
- closure/prototype behavior (10)

## Event Loop Correctness (20)
- queue semantics and ordering (15)
- starvation handling rationale (5)

## GC Correctness and Safety (20)
- reclaim behavior correctness (10)
- no obvious safety regressions (10)

## Instrumentation and Analysis (15)
- useful metrics and traces (10)
- analysis quality (5)

## Engineering Quality (10)
- tests, docs, architecture clarity

Pass thresholds:
- Minimum completion: >= 70
- Mastery completion: >= 85
