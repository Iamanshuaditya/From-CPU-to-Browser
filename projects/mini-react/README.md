# Project: Mini React (VDOM + Reconciliation + Hooks)

Build a small React-like runtime focused on correctness, scheduling, and measurable update costs.

## Scope
- VDOM representation
- diff + patch generation
- keyed reconciliation
- batched commits
- hooks baseline (`useState`, `useEffect`)

## Deliverables
- renderer + scheduler implementation
- hook lifecycle tests
- reconciliation correctness tests
- profiling report with commit/render time breakdown
