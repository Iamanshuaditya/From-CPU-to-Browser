# Mini React Milestones

## Milestone 1: VDOM + Initial Render
Exit criteria:
- simple component tree renders correctly.

## Milestone 2: Diff + Patch
Exit criteria:
- update tests pass for props/text/tree shape changes.

## Milestone 3: Keyed Reconciliation
Exit criteria:
- reorder tests preserve identity and behavior.

## Milestone 4: Hooks Baseline
Exit criteria:
- `useState` and `useEffect` tests pass.

## Milestone 5: Batching + Instrumentation
Exit criteria:
- measured reduction in unnecessary commits under burst updates.
