# Mini React Rubric

Scoring: 100 points.

## Reconciliation Correctness (30)
- diff accuracy (15)
- keyed reconciliation correctness (15)

## Hooks Correctness (25)
- `useState` behavior (10)
- `useEffect` lifecycle and cleanup (15)

## Performance and Scheduling (20)
- batched update behavior (10)
- render/commit measurement quality (10)

## Testing and Invariants (15)
- test suite depth and clarity

## Engineering Quality (10)
- architecture, readability, documentation

Pass thresholds:
- Minimum completion: >= 70
- Mastery completion: >= 85
