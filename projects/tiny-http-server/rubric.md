# Tiny HTTP Server Rubric

Scoring: 100 points total.

## Correctness (40)
- Request parsing correctness (15)
- Keep-alive semantics (10)
- Status code correctness (10)
- Protocol edge-case handling (5)

## Reliability (20)
- No crashes on malformed inputs (10)
- Timeout and connection cleanup behavior (10)

## Observability (15)
- Useful logs with correlation ids (8)
- Metrics endpoint and latency tracking (7)

## Performance (15)
- Baseline and degraded profile benchmarks (10)
- Analysis quality and bottleneck reasoning (5)

## Engineering Quality (10)
- Test coverage and clarity (5)
- Code organization and maintainability (5)

## Pass Thresholds
- Minimum completion: >= 70
- Mastery completion: >= 85
